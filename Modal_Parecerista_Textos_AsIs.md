## 1. `overlayExit`

ID do modal: `overlayExit`

Textos:

```text
Avaliação não finalizada
Sair e resetar esta avaliação?
Se você sair agora, tudo o que foi preenchido nesta ficha será descartado.
A avaliação só permanece no sistema depois do salvamento. Se quiser manter o que já fez, continue na ficha e conclua o envio.
Continuar avaliando
Sair e resetar
```

## 2. `overlayEarlySave`

ID do modal: `overlayEarlySave`

Textos:

```text
CONFIRMAÇÃO DE ENVIO
Concluir seu parecer agora?
Esta avaliação foi concluída em cerca de 2 minutos.
Análises completas desta ficha, com os 3 anexos, costumam levar em torno de 5 minutos. Se quiser revisar os critérios ou o parecer, a ficha continua aberta.
Observação adicional
(opcional)
Se quiser, registre uma observação adicional...
Continuar avaliando
Concluir e enviar
```

Texto dinâmico usado pelo JS no subtítulo:

```text
Esta avaliação foi concluída em cerca de ${elapsedMin} minuto${elapsedMin > 1 ? 's' : ''}.
```

## 3. `overlayRequiredParecer`

ID do modal: `overlayRequiredParecer`

Textos:

```text
PARECER GERAL
Justifique sua avaliação para concluir o envio
Como houve 3 ou mais marcações de Insuficiente, este campo deixa de ser opcional.
Por favor, registre uma justificativa geral do seu parecer para concluir e salvar a avaliação do material submetido.
Entendi
```

Texto dinâmico usado pelo JS no subtítulo:

```text
Como ${numInsuf} critérios foram marcados como Insuficiente, o parecer geral precisa ser preenchido para concluir e salvar esta avaliação.
```

## 4. `overlayNextEvaluation`

ID do modal: `overlayNextEvaluation`

Textos:

```text
Avaliação concluída
Deseja abrir a próxima avaliação agora?
Você pode seguir direto para a próxima ficha ou voltar para a lista.
Se continuar, a próxima avaliação pendente será preparada automaticamente no topo da tela.
Agora não
Sim, abrir próxima
```

## 5. `overlayMissingAttachments`

ID do modal: `overlayMissingAttachments`

Textos:

```text
ANÁLISE OBRIGATÓRIA
ENTREGÁVEIS (Vídeo + Roteiro 30s + Campanha)
Esta avaliação não pôde ser concluída porque ainda há materiais sem abertura registrada.
Para finalizar seu parecer, é necessário abrir e analisar os 3 entregáveis anexados.
Pendências
Entendi
```

Itens dinâmicos possíveis na lista de pendências:

```text
Vídeo criativo
Roteiro 30s
Campanha multiplataforma
```
